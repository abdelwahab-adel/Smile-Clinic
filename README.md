# 😊 Smile Clinic

**نظام إدارة عيادة طب أسنان حديث وسهل الاستخدام**

A modern, responsive web-based clinic management system built with cutting-edge web technologies.

## 📋 نظرة عامة | Overview

Smile Clinic هو تطبيق ويب متقدم مصمم لتسهيل إدارة عمليات العيادة والتفاعل مع المرضى. يوفر واجهة مستخدم نظيفة واحترافية مع تجربة مستخدم سلسة.

## 🚀 البدء السريع | Getting Started

### المتطلبات الأساسية | Prerequisites

- ✨ متصفح ويب حديث (Chrome, Firefox, Safari, Edge)
- 💻 لا توجد متطلبات تثبيت إضافية

### التثبيت | Installation

**الخطوة 1:** استنساخ المستودع
```bash
git clone https://github.com/abdelwahab-adel/Smile-Clinic.git
```

**الخطوة 2:** الانتقال إلى مجلد المشروع
```bash
cd Smile-Clinic
```

**الخطوة 3:** فتح التطبيق
- افتح ملف `index.html` في متصفح الويب الخاص بك
- أو استخدم خادم محلي (مثل Live Server)
